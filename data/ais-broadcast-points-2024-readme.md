**Title:** Automatic Identification System broadcast point data for U.S. waters 2024

- Daily Apache Parquet files hosted on Azure blob storage
- GeoParquet 1.0.0, Snappy compression
- Geometry type of "Point" in well-known binary (WKB) encoding
- World Geodetic System 1984 coordinate reference system
- Coordinated Universal Time - UTC
- Downsampled to the nearest whole minute
- Maximum file size approximately 325 MB

**Citation:** Martin, Daniel R., Jesse Brass, Matthew Dornback, and Jeremy Fontenault. 2025. Nationwide Automatic Identification System 2024. NOAA Office for Coastal Management.
