**Title:** Automatic Identification System vessel track data for U.S. waters 2024

- Monthly Apache Parquet files hosted on Azure blob storage
- GeoParquet 1.1.0, ZSTD compression
- Geometry type of "LineString" in well-known binary (WKB) encoding
- World Geodetic System 1984 coordinate reference system
- Coordinated Universal Time - UTC
- Downsampled to the nearest whole minute
- Maximum file size approximately 1.78 GB

**Citation:** Martin, Daniel R., Jesse Brass, Matthew Dornback, and Jeremy Fontenault. 2026. Nationwide Automatic Identification System 2024. NOAA Office for Coastal Management.
